Store Landing Page Configuration Examples

This page demonstrates how to configure the store landing page to disable images and reviews using data attributes.

Example 1: Default Configuration (Images and Reviews Enabled)

HTML Configuration:

<div id="pmt-store-landing-page-container" data-api-url="https://public-api.test.pinmeto.com/pinmeto/abc123/locations.json"> </div>

Result: Both images and reviews will be displayed.

Example 2: Disable Images Only

HTML Configuration:

<div id="pmt-store-landing-page-container" data-api-url="https://public-api.test.pinmeto.com/pinmeto/abc123/locations.json" data-show-images="false"> </div>

Result: Images will be hidden, reviews will still be displayed.

Example 3: Disable Reviews Only

HTML Configuration:

<div id="pmt-store-landing-page-container" data-api-url="https://public-api.test.pinmeto.com/pinmeto/abc123/locations.json" data-show-reviews="false"> </div>

Result: Reviews will be hidden, images will still be displayed.

Example 4: Disable Both Images and Reviews

HTML Configuration:

<div id="pmt-store-landing-page-container" data-api-url="https://public-api.test.pinmeto.com/pinmeto/abc123/locations.json" data-show-images="false" data-show-reviews="false"> </div>

Result: Both images and reviews will be hidden.

Complete Example with All Configuration Options

HTML Configuration:

<div id="pmt-store-landing-page-container" data-api-url="https://public-api.test.pinmeto.com/pinmeto/abc123/locations.json" data-google-maps-api-key="your-google-maps-api-key" data-default-image-url="https://example.com/default-store-image.jpg" data-store-locator-url="https://example.com/store-locator" data-home-url="https://example.com" data-locales-path="locales/" data-css-path="/css/simple-landing-page.css" data-show-images="false" data-show-reviews="false"> </div>

Result: A fully configured landing page with images and reviews disabled.

Configuration Options Reference

Attribute Default Description
data-show-images "true" Set to "false" to hide the photo album section
data-show-reviews "true" Set to "false" to hide the reviews widget

JavaScript Configuration (Alternative Method)

You can also configure these options via JavaScript:

// Set before the script loads window.SHOW_IMAGES = false; window.SHOW_REVIEWS = false;

Note: This method requires setting the variables before the landing page script loads.
